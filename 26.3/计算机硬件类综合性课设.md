# 计算机硬件类综合性课设

## 一.引言
### 1.用什么做？
ASIC：专用集成电路，类比于成品房，可扩展性较差。
PFGA：现场可编程门阵列，类比于砖瓦等建筑材料，可扩展性较好。

SoC：System on Clip

AP SoC：All Programmable System-on-Chip

Xilinx AP SoC--Zynq
Processing System（PS）：包含多核ARM处理器
Programmable Logic（PL）：一块FPGA
PS和PL通过AXI走线（AXI interface）连接

PL的基本组成资源单位：
  CLBs
    LUT：根据索引直接输出值，组合逻辑电路
    FF（Flip-Flot）：触发器，时序电路

  IOBs：
  Block RAM
  DSP

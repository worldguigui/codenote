# 三维模型制作

三维建模的入门课，掌握基本的模型制作即可。<br/>

使用工具：3ds_Max。<br/>

布局介绍：打开3ds_Max，中间区域的四个窗口，代表着模型的四个视图。<br/>
右侧是命令面板，负责对模型进行更精细的调整。<br/>
下方有带有刻度的时间条，用来进行动画的制作。<br/>
右下方有八个小按钮，可以对视图的视野进行放大或缩小等操作。<br/>
使用快捷键L、R、F可以分别恢复左视图、右视图、前视图。<br/>
上方是菜单栏，可以进行各种操作。<br/>
按名称选择：按照名称选取已创建的模型。<br/>
矩形选择区域：可以选中矩形、多边形等框住的模型。<br/>
窗口/交叉：窗口模式表示要完整框住模型才可以选中，交叉模式表示框住模型的部分即可选中。<br/>

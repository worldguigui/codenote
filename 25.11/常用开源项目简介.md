# 介绍一些Java后端学习中可能遇到的比较基础的开源项目

## 一. Redis

### 1.什么是Redis
Redis 是一个开源的、基于内存的、可持久化的键值对存储系统。它通常被用作数据库、缓存和消息中间件。<br/>
开源意味着我们可以找到Redis的源码。<br/>
基于内存意味着Redis可以直接访问存储在内存（RAM）中的数据，读写无需磁盘I/O，因此性能非常高，通常能达到每秒数十万次的读写。<br/>
可持久化意味着Redis可以将内存中的数据异步保存到硬盘上，防止服务器断电或重启时数据丢失。<br/>
### 2.Redis的数据类型
Redis以键值对(key:value)的形式存储数据，而value支持非常丰富的数据类型。<br/>
包括字符串String，哈希Hash，列表List，集合Set，有序集合Sorted Set以及其他更复杂的类型。<br/>

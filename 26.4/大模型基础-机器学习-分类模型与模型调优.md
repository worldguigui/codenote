# 大模型基础-机器学习-分类模型与模型调优

了解完回归模型后，我们再来了解另一种常用的模型-分类模型。

现在有这么一个情景：给你肿瘤的大小作为特征，预测这个肿瘤是恶行还是良性的。

如果继续使用回归模型进行预测，训练样本的目标变量只有两种值：0或1，回归模型会预测出0~1附近的无穷多的值，但我们只需要知道预测出的分分类：恶性（用1表示）还是良性（用0表示）

这时候我们尝试选择一个阈值0.5，如果预测值超过这个阈值，则认为它是恶性的，反之，则是良性的。

阈值与最拟合的直线在某处相交（假如说记为x=200），对横轴作垂线，可以认为：左侧的样本预测值都是良性的，右侧都是恶性的。

这条直线我们可以叫它分界线，也成为决策边界（decision boundary）。

这时候就有一个问题：如果引入更多的训练样本，最拟合直线也可能因此而改变，导致决策边界偏移。但实际上，添加数据不应改变我们关于分类恶性还是良性的任何结论。

恶性就是恶性，不能因为有很多尺寸更大的恶性案例存在，就认为恶性不是恶性了。

因此，我们可以认为，线性回归已经不适合来解决这种问题。我们需要世界上使用最广泛的分类算法--逻辑回归（logic regression）。


## 一.逻辑回归
逻辑回归虽然名字里带着回归，但其实是分类算法（历史遗留问题）。

既然是另一种算法，我们就要用全新的曲线、全新的代价函数等来描述它。

### 1.逻辑函数
逻辑函数（Sigmoid function），$g(z) = \frac1{1+e^{-z}}$ ，针对分类问题，逻辑回归会拟合一条这样的曲线。

最终得到$f_{w,b}(x) = g(w·x + b)$。

### 2.决策边界
逻辑函数只会给我们输出0~1之间的无限的连续值，我们需要凭借我们强大的数学直觉定义一个值，来决定预测结果最终会映射到哪个分类。

### 3.成本函数
Loss/L：损失函数，用来衡量单个样本预测值和目标值的偏差。

成本函数J就是所有样本的L的总和。

在线性回归模型中，我们的L是预测值与目标值差值的平方，J是平方误差代价函数。

但这个L和J对于分类问题并不适用。

平方误差损失函数通过计算预测值和目标值差值的平方来衡量它们之间的偏差，直觉上可以看出来，偏差越大，说明模型预测的结果越差。

对于分类问题，虽然平方误差可以衡量预测值与目标值的差距，但在结合 sigmoid 函数后，会导致一个严重问题：当预测值接近 0 或 1 时，sigmoid 的导数（p*(1-p)）变得很小，使得梯度被压缩。

这样一来，即使预测结果非常错误（例如 0.001 相比 0.01 更离谱），其对应的梯度反而更小，模型参数更新更慢，从而影响训练效果。

我们要重新定义一个损失函数用来衡量偏差：

$L = -log(f_{w,b}(x^{(i)}))$ ， when y^i = 1

$L = -log(1-f_{w,b}(x^{(i)})) $ ， when y^i = 0

也可以将上面式子中的条件融合到式子里面：

$L = -y^{(i)}log(f_{w,b}(x^{(i)})) - (1 - y^{(i)})log(1-f_{w,b}(x^{(i)}))$

将所有预测结果的损失函数值相加，就可以得到代价函数。

$
J(w,b) = -\frac1m\sum_{i=1}^{m}[y^{(i)}log(f_{w,b}(x^{(i)})) + (1 - y^{(i)})log(1-f_{w,b}(x^{(i)}))]
$

### 4.梯度下降
和前面讲到的线性回归模型中的梯度下降一样，尝试用梯度下降算法来找到最小值。

每轮更新中，w和b减去代价函数对应点的偏导数值与学习率α（learing-rate）的乘积。

也就是：$w = w - α * \frac{dJ(w,b)}{dw}$ 和 $b = b - α * \frac{dJ(w,b)}{dw}$

（这里应该是偏导，但我没找到对应的符号......）

$$
\frac{dJ(w,b)}{dw} = \frac1m\sum_{i=1}^{m}(f_{w,b}(x^{(i)}) - y^{(i)}) x^{(i)}
$$

$$
\frac{dJ(w,b)}{db} = \frac1m\sum_{i=1}^{m}(f_{w,b}(x^{(i)}) - y^{(i)}) 
$$

你也许会发现，这些内容不是和线性回归那一部分一模一样吗？

是这样的，前面也说过，梯度下降算法是种普适的算法，不仅仅用于线性回归问题。

唯一不同的地方是线性回归模型的函数是$f_{w,b}(x) = w · x + b$，而逻辑分类模型的函数是$f_{w,b}(x) = \frac1{1 + e^{-(w·x + b)}}$

## 二.模型调优
到目前为止，我们已经学习了解决回归问题和分类问题的两种模型，它们在许多任务中表现良好，但在某些应用中，算法可能会遇到一些问题。

### 1.欠拟合、泛化、过拟合
当模型无法很好的拟合到训练样本上（如用线性回归拟合二次函数），我们称之为欠拟合（underfitting），也可以说该算法具有高偏差（high bias）。

模型不只是要在训练样本上拟合的很好，同时对于训练样本之外的数据的预测结果也很准，我们称该算法有很好的泛化能力（generalization）。

我们可以选取多个参数w，用复杂的高次多项式来拟合数据。模型可以很好地拟合到训练样本上，但它的泛化能力很差，我们称之为过拟合（overfitting）。也可以说该算法具有高方差（high variance）。

过拟合的算法很努力地拟合每一个训练样本，但当新样本有一点点不同，它的预测结果就天差地别。

### 2.解决过拟合
一，获取更多训练样本可以解决过拟合问题。

二，如果训练样本数无法增加，选择减少特征数量（n）。特征越多，需要的样本数量就应该越多。

三，正则化（regularization），消除特征可以看作将某个特征的参数w置为0，正则化的思想与此类此：保留这个特征，但让它的参数w尽可能小，无法产生过大的影响。

### 3.带正则化的成本函数
现在有一个四次函数回归算法，$w_1x + w_2x^2 + w_3x^3 + w_4x^4 + b$。

它的成本函数是：$J(w,b) = \frac{1}{2m}\sum_{i=1}^{m} (yhat^{(i)} - y^{(i)})^2$。

如果在成本函数中额外加入两个项：$1000w_3^2$和$1000w_4^2$，这意味着在训练时，如果w3和w4的取值过大，你就会狠狠惩罚这个模型（因为成本太高了）。

这样模型就倾向于让w3和w4趋于0，这样就相当于我们消除了这两个特征。

正常情况下，我们不能确定具体应该惩罚哪个参数（即哪个参数更重要），所有应该有个更一般化的方法：$J_{w,b}(x) = ... + \fracλ{2m}\sum_{j=1}^nw_j^2$

新的代价函数由原始代价函数和正则项两部分构成。

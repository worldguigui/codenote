# Java基础-面向对象编程

## 一.引言
封装、继承、多态是面向对象编程（Object-Oriented Programming，OOP）的三大特性。

Java是一门典型的面向对象的语言。

## 二.封装
将属性和方法封装在类的内部，对外隐藏内部细节，只暴露方法给外部调用。

通过权限控制符（private、publi、protected）来控制外部对内的可见程度。

private：仅类内部可以访问。

default（默认）：同一包的类可以访问。

protected：子类和同一包的类可以访问。

public：公开，允许所有类访问。

## 四.继承
继承是子类复用父类属性和方法的一种机制，通过 extends 实现。

子类会继承父类所有非私有的属性和方法（不包括构造方法）。

Java中，只支持单继承，和链式继承。不允许多继承。

```java
// ×
class A extends B,C{}
// √
class A extends B{}
class B extends C{}
```

### 1.this和super
this指向当前对象，super指向父类/超类对象

---
this() 规则：

可以出现在任何方法/构造中，但必须在第一行。

不能和super同时出现。（因为不可能让它们同时在同一行）

创建对象时，默认调用无参构造；若类中没有无参构造，使用默认的无参构造。

---

super() 规则：

可以出现在任何方法/构造中，但必须在第一行。

如果子类构造方法没有写 super()，编译器会自动加 super()（调用父类无参构造）

如果父类没有无参构造，子类必须手动调用 super(参数)

### 2.方法重载和重写
方法重载，允许名称相同、但参数列表不同的方法。

---

方法重写：

子类可以修改继承自父类的方法的逻辑，名称和参数列表要保持不变

可以选择用注解 @Override 标识。

重写的方法的可见程度不能比父类更严格。

不能抛出父类没有声明的检查型异常。

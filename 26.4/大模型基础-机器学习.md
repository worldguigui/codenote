# 大模型基础-机器学习-引言与回归模型

## 一.引言
什么是机器学习？从字面上理解，就是赋予计算机在没有明确的编程情况下，仍然可以学习的能力。

为什么要学习机器学习？这与大模型的联系是什么?

机器学习是大模型的基础，在学习完机器学习的相关知识后，我们才具备最基本的操作大模型的能力，而不是跟着AI一次次盲目的重试、调参。

机器学习算法最核心的分类：监督学习（Supervised Learning）和无监督学习（Unsupervised Leaning）。

监督学习应用最广，也是与大模型关联最多的分类，所以，主要掌握这方面的知识。

## 二.监督学习
监督机器学习是学习从X到Y（或是从输入到输出）之间映射关系的算法。

给予算法正确的示例，也就是输入X和期望的正确的输出标签Y，从而让算法可以在只给出输入X的情况下，给出相当准确的预测。

监督学习中有两种任务类型：回归算法和分类算法

### 1.回归算法
回归（Regression）算法是一种用来预测连续数值的算法，从无限可能输出的数字中预测数值。

比如说预测房价、预测销量等。

### 2.分类算法
分类（Classification）算法是一种用来预测离散数值的算法，从一部分可能的输出中预测一个类别。

分类可以是数字，如1，2；也可以是“猫”、“狗”。

## 三.无监督学习
无监督学习，只提供输入X，由算法自己在这些数据中找到某种结构、模式或有趣的东西。

### 1.聚类算法
聚类算法（Clustering）：将相似的数据点分到同一组，也叫做同一簇（cluster）。

### 2.异常检测
异常检测（Anomaly detection）：找到不正常的数据点。

### 3.降维
降维（Dimensionality reduction）：将大数据集几乎神奇地压缩为小得多地数据集，同时尽可能少地丢失信息。

## 四.线性回归模型
线性回归模型（Linear Regression Model）：将一条直线拟合到数据上，这是最简单、基础的模型，其相关的许多概念也适用于其他机器学习模型。

### 1.常用的符号表示
巡训练集（training set）：用来训练模型的数据集（data set）。

输入变量（input variable）：也叫做特征（feature）或输入特征，用小写x表示。

输出变量（output variable）：也叫做目标变量（target），用小写y表示。

训练样本总数（training examples）：用小写m表示。用(x，y)表示一组训练样本（training example）

第i个训练样本：用带有上标的(x，y)表示样本的序号。x^(2),y^(2)表示第二个训练样本数据，而不是它们的平方值。

预测值（prediction/estimated y）：算法根据输入值x预测出的可能结果，用 y-hat 表示。

模型参数（model parameters）：在线性回归模型中，算法可以表示为 y=wx+b，它们又可以被叫做系数（coefficients）或权重（weights）。在机器学习中，模型参数是你在训练过程中可以调整的的变量，以改进模型。

### 2.如何实现线性回归
#### （1）成本函数的定义
成本函数/代价函数（cost function）可以告诉我们模型的表现如何，以便我们可以去优化它。

在线性回归模型中，常使用平方误差代价函数（Squared error cost function）。

代价函数通过计算预测值 y_hat 与目标值 y 的差值，来衡量预测值与目标值之间的偏差。

对所有差值求和，计算出整个训练集上的误差。

为了不让误差和随着训练集规模地增加而自动变大，对其求均值。

为了方便后续机器学习过程地计算，代价函数整体除以2，这一步实际上并不影响它的有效性。

J(w,b) = 
$$
\frac{1}{2m}\sum_{i=1}^{m} (yhat^{(i)} - y^{(i)})^2
$$

我们最终希望找到一对 (w,b) 使得代价函数的值最小，这意味着整体的预测值和目标值的误差最小。

到现在我们还只是通过穷举来寻找最小代价对应w，b的值，事实上，我们还可以通过梯度下降算法（gradient descent）来系统地找到它们。

### 3.梯度下降算法
梯度下降算法广泛应用在机器学习的领域中，不仅是线性回归模型，还用来训练复杂的神经网络模型，也称为深度学习。

梯度下降算法是一种可以用来尝试最小化任何函数的方法，而不仅仅是线性回归。

在算法的开始，对于代价函数J(w,b)，我们要从对w，b的初始猜测开始（初始值是什么并不重要，一般设为0），每次稍微改变参数w和b的值，使得J(w,b)变小，最终稳定在极小值。

每轮更新中，w和b减去代价函数对应点的偏导数值与学习率α（learing-rate）的乘积。

也就是：$w = w - α * \frac{dJ(w,b)}{dw}$ 和 $b = b - α * \frac{dJ(w,b)}{dw}$

（这里应该是偏导，但我没找到对应的符号......）

$$
\frac{dJ(w,b)}{dw} = \frac1m\sum_{i=1}^{m}(f_{w,b}(x^{(i)}) - y^{(i)}) x^{(i)}
$$

$$
\frac{dJ(w,b)}{db} = \frac1m\sum_{i=1}^{m}(f_{w,b}(x^{(i)}) - y^{(i)}) 
$$

在数学之外有个很浪漫的类比：你站在一座山山顶，想尽可能快得到达山谷的最低处，怎么做？你选择环顾四周，从高度下降得最快的方向下山；你每迈一步，都会重新选择下降的方向。

在数学中，导数对应的就是高度下降最快的方向。

学习率α就是你每次迈步的步长，步长太短，你需要非常多时间才能到达极值点；步长太长，你可能直接跳过极值点，在其附近摇摆。

w、b决定了你所在的位置（在三维空间中）。

上面提到的这种梯度下降算法被称为批量梯度下降算法（batch gradient descent），在每次梯度下降的过程中，我们都要查看所有的训练样本，而不是训练集的一个子集。

## 五.多特征的线性回归模型
### 1.常用的符号表示
在最初的线性回归中，我们输入房子大小及其对应的价格，以此来根据大小来预测价格。

但价格还可能与房子大小、楼层、房间数、房龄有关。所以我们使用多特征线性回归模型（multiple linear model），考虑多个特征，来预测房子的价格。

x_(j) 表示第j个特征，如房子大小、楼层等。

n 表示特征的数量。

x^(i) 仍然表示第i个训练样本，只不过这里的样本是包含多个特征值得向量，如 x^1 =（120, 6, 3, 20)。 

代价函数也要发生改变：$f_{w,b}(x) = w_1x_1 + w_2x_2 + w_3x_3 + ... + w_nx_n + b$

为了简化表示，我们使用点乘表示：$f_{w,b}(x) = w · x + b$，其中w、b皆是向量。

### 2.向量化
使用向量化可以让我们的代码更简洁、计算速度更快。

Python中的Numpy类库提供了向量的初始化以及计算的方法。

```python
# init
w = np.array([1,0,2.5,-3.3])
b = 4
x = np.arr([10,20,30])

# 计算代价
f = np.dot(w,x) + b

# 梯度下降
w = w - 0.1 * dw
```

### 3.梯度下降算法
与单一特征的线性回归梯度下降算法的核心思想一样，也要计算$w_j = w_j - α * \frac{dJ(w,b)}{dw}$ 和 $b = b - α * \frac{dJ(w,b)}{dw}$

不同的是，由于有多个特征，需要更新从1到n的所有参数w：
$$
\frac{dJ(w,b)}{dw_1} = \frac1m\sum_{i=1}^{m}(f_{w,b}(x^{(i)}) - y^{(i)}) x^{(i)}_1
$$

$$
\frac{dJ(w,b)}{dw_2} = \frac1m\sum_{i=1}^{m}(f_{w,b}(x^{(i)}) - y^{(i)}) x^{(i)}_2
$$
$$\vdots$$
$$
\frac{dJ(w,b)}{dw_n} = \frac1m\sum_{i=1}^{m}(f_{w,b}(x^{(i)}) - y^{(i)}) x^{(i)}_n
$$

b的计算没有太大的变化，甚至可以说是没有变化（计算过程中的w，一个值->一个向量）：

$$
\frac{dJ(w,b)}{db} = \frac1m\sum_{i=1}^{m}(f_{w,b}(x^{(i)}) - y^{(i)}) 
$$

### 4.特征缩放
#### （1）为什么
假如我们要通过房子大小x1和楼层x2推测房间价格，x1的取值在20~200，x2的取值在1~5。

可以看到，x1的范围比x2大得多。

这就导致绘制等高线图时，得到一个比例很不协调的椭圆（长轴较长，短轴较短）。

在梯度下降的过程中，走“之”字形路线，沿着长轴震荡，导致要花费更多的步数才能得到最佳结果。

这时候我们就要用到特征缩放，让所有特征都处在相近的量级。

#### （2）怎么做
保证训练数据的范围在-6~6之间即可，有一定偏差也无所谓，但若是太小（-0.001~0.002）或太大（2000~30000）的话，推荐进行特征缩放：

**直接除：**

取每个原始的x1值并除以范围的最大值。

**均值归一化（Mean normalization）：**

计算特征1上数据的均值μ1，然后取每个原始的x1值，减去μ1，最后除以范围大小（范围最大值-范围最小值）。

**Z分数归一化（Z-score normalization）：**

计算每个特征的均值μ1和标准差σ1，然后取每个原始的x1值，减去μ1，最后除以σ1。

### 5.学习率的选择
我们可以以迭代次数（iterations）为横轴、代价函数值为纵轴绘制曲线，用来检查梯度下降是否收敛。

正常的梯度下降过程，曲线会随着迭代次数先下降，随后保持不变。

如果曲线忽上忽下，可能是学习率设置得过大，导致来回震荡；

如果曲线变化缓慢，可能是学习率设置得过小；

这背后的原因很符合直觉，就不过多解释了。

自动收敛测试：也可以在训练过程中，设置一个ε，当相邻两轮迭代的代价函数值变化小于ε时，结束训练。

## 六.多项式回归
随着问题变得复杂，单纯的线性回归已经无法很好得拟合到训练数据上了，这时候我们需要非线性回归算法-多项式回归（Polynomial Regression），它结合了多元线性回归和特征工程的思想。

### 1.特征工程
考虑一个情景：我们正在预测房价，已知该房子的长和宽。

原始特征（长、宽）：$f_{w,b}(x) = w_1x_1 + w_2x_2 + b$

我们可以将长或宽作为特征来预测，但我们可能会想，房子的占地面积（长*宽）作为特征会不会更合适？

新增一个特征（长*宽）：$f_{w,b}(x) = w_1x_1 + w_2x_2 + w_3x_3 + b$

这就是特征工程，通过转换或组合原始问题的原始特征来设计一个新特征。

我们可能得到一个更好的模型，也可能不。

### 2.多项式回归
$f_{w,b}(x) = w_1x + w_2x^2 + w_3x^3 + b$

我们想要根据房子的面积x来预测房价y。

n=3，特征分别是面积的一次方、面积的平方、面积的立方。

当然，由于平方和立方扩大了训练数据的取值范围，最后我们要进行适当的特征缩放。
